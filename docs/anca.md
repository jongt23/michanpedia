# Ancá
## Mejor con tílde para no confundir con anca de rana...

Sirve para múltiples situaciones y propósitos.

- En casa de...
- A casa de...
- En la casa de...
- A la casa de...

Ejemplo: "Ancá mi tío" 

Pero si tienes que usar "el/del" o similares, se usará "Ancar" por ejemplo: 

- "En la casa del primo" sería: **"Ancar** primo"

Por supuesto si en lugar de una casa particular es un cortijo, una tienda, un comercio, un estercolero, en fin, lo que sea, también vale "ancá" igualmente.

- "En el Kiosco del Koki": Aquí tenemos el comercio y el **del** antes del nombre y sería

    - "Ancar Koki"
